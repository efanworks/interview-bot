# Interview Bot
输入：岗位描述 + 简历/项目描述
输出：面试问题预测

## 技术栈
- React 19
- TypeScript
- Vite 5

## 开发

### 安装依赖
```bash
npm install
```

### 开发服务器
```bash
npm run dev
```
打开浏览器访问 http://localhost:3000

### 构建生产版本
```bash
npm run build
```

### 预览构建结果
```bash
npm run preview
```

### 类型检查
```bash
npm run type-check
```
